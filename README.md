## 📖 About the Project

The **Malabar Table** is a responsive restaurant reservation system developed to simplify online table booking and restaurant management.

Customers can browse the restaurant, explore the menu, view galleries, register/login, and reserve tables online. The administrator can manage reservations, customers, and restaurant data through a dedicated admin panel.

The application focuses on providing a smooth user experience while reducing manual reservation handling.

## 🚀 Getting Started

### Prerequisites

- PHP 8+
- MySQL
- XAMPP

---

### Installation

Clone the repository

```bash
git clone https://github.com/auyashas/restaurant-application.git
```

Move the project into

```
xampp/htdocs/
```

Start

- Apache
- MySQL

Open

```
http://localhost/restaurant-application
```

---

## 🗄 Database Setup

1. Open phpMyAdmin

2. Create a database

```
restaurant_db
```

3. Import

```
restaurant_db.sql
```

4. Run the application.
